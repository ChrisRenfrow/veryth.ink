# Syntax Highlighting

## Bash
```sh
function say_hello {
    echo "Hello, $1"
}
```

## JavaScript
```js
function say_hello(name) {
    console.log(`Hello, ${name}`);
}
```

## Rust
```rust
fn say_hello(name: &str) {
    println!("Hello, {}", name);
}
```
